LESS - css preprocesszor
Vissza a főoldalra
1. Változók:
Újrafelhasználható értékek, pl. színek, mértékek.
@lt-gray: #ddd;
@background-dark: #512DA8;
h1 {
  color: @lt-gray;
  background: 
  @background-dark;
}
2. Mixinek: újrafelhasználható stíluscsoportok. 
.zero-margin {
  margin: 0 auto;
  background: white;
}
.row-header {
  margin: zero-margin;
  padding: 0 auto;
}
3. Beágyazás: CSS hierarchikus struktúrában, tükrözve a HTML-t.
@background-dark: #512DA8;
.carousel {
  background: @background-dark;
  .carousel-item {
    height: 300px;
    img {
      position: absolute;
      top: 0; left: 0;
      min-height: 300px;
    }
  }
}
4. Matematikai műveletek 
@carousel-item-height: 300px;

.carousel-item {
  height: @carousel-item-
  height;
}
.carousel-item .item-large {
  height: @carousel-item-
  height * 2;
}
5. Függvények: matematikai, színkezelési, listázási műveletek.
@base: 0.5;
@width: 0.8;

.class {
  width: percentage
  (@width);
  color: saturate
  (@base, 5%);
  background-color: 
  light(@base, 25%);
}
